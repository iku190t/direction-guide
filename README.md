# 方向ガイド

iPhone Safariで使う方向案内の試作アプリです。

## 使い方

1. iPhone SafariでHTTPSの公開URLを開く
2. `センサー開始` を押す
3. 機械の方向に向けて、必要なら二本指で目盛りを回して0度を合わせる
4. 一本指で青い仮目標を合わせる
5. `目標をセット` を押す
6. 歩きながら中央の `右 2°` / `左 5°` を見る

## 表示

- 赤い `N`: 北
- 緑の針: 現在方向
- 青い針: 仮目標
- 黄色の針: セット済み目標

## 注意

iPhone Safariの方位センサーは、基本的にHTTPSページでないと動きません。
