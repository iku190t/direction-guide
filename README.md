# 方向ガイド

iPhone Safariで使う方向案内の試作アプリです。

## 使い方

1. iPhone SafariでHTTPSの公開URLを開く
2. `センサー開始` を押す
3. 機械の方向に向けて `今の機械方向を0°にする` を押す
4. コンパスをタップ/なぞって黄色の目標線を合わせる
5. 歩きながら `右 2°` / `左 5°` を見る

## カメラで合わせる

1. `カメラで合わせる` を押す
2. 中央の黄色線を目標物に合わせる
3. `この方向を0°` または `黄色線にセット` を押す
4. `閉じる` で通常画面に戻る

## 表示

- 周りの数字: 機械方向を0°にした相対目盛り
- 周りの数字: 現在方向に合わせて動く
- 黄色の線: 目標方向

## 注意

iPhone Safariの方位センサーは、基本的にHTTPSページでないと動きません。
